Key Project

Real-Time Anomaly Detection in Sensor Data

A robust LSTM Autoencoder to detect anomalies in high-frequency time-series data for predictive maintenance.

Role & Tech Stack: Solo Developer | Python, PyTorch, FastAPI, Docker

Approach:

  • Engineered a data pipeline to process and normalize a 10GB sensor dataset.
  • Designed and trained an LSTM-based autoencoder model to learn normal operational patterns.
  • Identified anomalies by flagging data points with high reconstruction error.
  • Developed a REST API using FastAPI for real-time inference and model deployment.
  • Containerized the application with Docker for portability and scalability.

Outcome & Metrics:

Achieved a 98.7% F1-score in identifying critical anomalies, outperforming a baseline statistical model by 15%. Average inference latency is under 50ms per request.

What I Learned:

This project deepened my understanding of unsupervised learning for time-series data and the practical challenges of deploying ML models. I focused on writing clean, modular code and ensuring the inference endpoint was both fast and reliable.
